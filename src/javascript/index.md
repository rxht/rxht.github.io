# Javascript
 javascript 是一种脚本语言，它可以在浏览器中运行。

## 浏览器中的 javascript
 浏览器中的 javascript 是一种解释型语言，它可以直接在浏览器中运行。

### 浏览器中的 javascript 运行原理
 浏览器中的 javascript 运行原理是：浏览器会将 javascript 代码解析成 AST 抽象语法树，然后将 AST 抽象语法树编译成字节码，最后浏览器会将字节码解释成机器码并执行。
