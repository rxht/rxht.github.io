# 导出 Atom Index Range

导出功能主要模块

![Export Atom Index Range](./assets/export_index_range.png)


## 操作步骤


1. 点击打开工具栏按钮，在主视图中显示工具栏
2. 在工具栏中最左侧功能 **粒度选择** 点击下拉，并选择对应的 **粒度** 此处选择 **Atom/Coarse Element Instances** 这个选项
3. 在主视图中选择需要导出的原子
4. 再主界面中点击鼠标右键，选择 `Export Index Range` 即可完成下载保存
