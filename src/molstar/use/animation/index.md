# 动画功能

当前所支持的动画支持如下：

1. **Animate Trajectory**：
    - 轨迹动画，可配置动画模式【往返、循环、一次】、持续时间、FPS
    - TIP: 文件必须要求带有轨迹信息

2. **Camera Spin**：<Badge type="tip" text="all" />
    - 相机旋转，可配置时长、速度、方向
    
3. **Camera Rock**：<Badge type="tip" text="all" />
    - 相机左右摇摆，可配置时长、速度、角度
