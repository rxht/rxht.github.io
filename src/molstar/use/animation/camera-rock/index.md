# Camera Rock

## 以.xyz 文件为例查看相机左右摇摆动画

1. 在 Qbics-MolStar 界面打开或拖入 .xyz 文件 显示渲染。

![file](./assets/file.webp)

2. 击图形界面左上角 Select Animation 按钮，选择不同的轨迹或场景动画演示方式。在此处，我们选择 Camera Rock 方法（相机左右摇摆动画）

![camera-spin](./assets/camera-rock.webp)
