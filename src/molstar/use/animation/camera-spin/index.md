# Camera Spin

## 以.xyz 文件为例查看相机旋转动画

1. 在 Qbics-MolStar 界面打开或拖入 .xyz 文件 显示渲染。

![trajectory](../../../tutorial/assets/trajectory.webp)

2. 击图形界面左上角 Select Animation 按钮，选择不同的轨迹或场景动画演示方式。在此处，我们选择系统默认的 Camera Spin 方法（相机旋转动画）

![camera-spin](./assets/camera-spin.webp)

3. 动画播放效果如下：

![camera-spin](./assets/camera-spin.gif)
