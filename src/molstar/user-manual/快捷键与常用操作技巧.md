# 十二、快捷键与常用操作技巧

本章汇总平台核心操作快捷键、操作速查表及科研场景下的高效使用技巧，帮助科研人员提升分子结构可视化、分析的操作效率，减少重复操作，聚焦科研核心分析工作。

## 1. 操作速查表

整合科研常用核心操作，按功能分类整理，便于快速查阅、即时调用，适配日常结构查看、分析的高频需求。

|操作功能|快捷键/操作方式|适用场景|
|---|---|---|
|视图旋转|鼠标左键拖动|调整分子结构角度，观察不同方位细节|
|视图平移|鼠标右键拖动 或 鼠标左键+Ctrl键拖动|移动分子结构位置，避免关键区域遮挡|
|视图缩放|鼠标滚轮/中键滚动|放大观察活性位点、原子细节，缩小查看整体结构|
|视图剖切|鼠标滚轮/中键滚动 + Shift键|隐藏分子表面结构，查看内部空腔、结合界面|
## 2. 键盘快捷键汇总

按操作功能分类，详细列出所有键盘快捷键及对应操作，包含视图控制、移动控制、进阶控制等，适配不同操作习惯，兼顾鼠标与键盘协同操作效率。

### 2.1 视图控制快捷键

- 旋转视图：鼠标左键拖动

- 绕Z轴旋转（滚动）：鼠标左键 + Ctrl键 + Shift键 拖动

- 平移视图：鼠标右键拖动 或 鼠标左键 + Ctrl键 拖动

- 聚焦结构：三指拖动

- 聚焦并缩放：鼠标滚轮/中键拖动

- 缩放视图：鼠标滚轮/中键滚动

- 剖切视图：鼠标滚轮/中键滚动 + Shift键

### 2.2 移动控制快捷键

- 向前移动：W键 或 游戏手柄上键

- 向后移动：S键 或 游戏手柄下键

- 向左移动：A键

- 向右移动：D键

- 向上移动：R键

- 向下移动：F键

- 加速移动：左Shift键

### 2.3 进阶控制快捷键

- 向左滚动：Q键

- 向右滚动：E键

- 向上倾斜：上箭头 + Shift键

- 向下倾斜：下箭头 + Shift键

- 向左偏航：左箭头 + Shift键

- 向右偏航：右箭头 + Shift键

- 启用指针锁定：Space键 + Ctrl键

## 3. 高效查看与分析小技巧

结合科研实际使用场景，汇总实用操作技巧，帮助科研人员减少操作步骤、提升分析效率，适配结构查看、细节分析、成果展示等核心需求。

### 3.1 视图优化技巧

- 快速聚焦活性位点：选中活性位点任意残基或原子，右键选择“居中显示”，再通过滚轮放大，可快速定位并查看细节，避免反复调整视图。

- 避免视图遮挡：分析配体与受体结合模式时，先隐藏水分子、无关离子，再使用“剖切”功能（滚轮+Shift键），可清晰查看配体与活性位点的结合界面，无多余结构干扰。

- 固定常用视图：对于常用的结构角度，调整好后保存会话（Ctrl+S），后续无需重新调整，直接加载会话即可快速调用，适用于多次重复分析同一结构的场景。

### 3.2 选择与分析技巧

- 批量选择高效操作：按住Ctrl键，鼠标左键依次单击目标原子、残基或链，可实现多片段批量选中；若需选中除目标结构外的所有片段，使用“反选”功能，减少重复选择操作。

- 测量结果快速导出：完成距离、角度等测量后，直接点击“导出所有测量”，选择CSV格式，可快速将数据导入Origin、GraphPad Prism等软件进行统计分析，无需手动记录。

- 二级结构快速筛选：激活二级结构分析功能（S键）后，可单独勾选α-螺旋、β-折叠等类型，快速隐藏无关二级结构，聚焦目标结构区域的分析。

### 3.3 成果展示与操作效率技巧

- 快捷键组合使用：结合“W/A/S/D/R/F”移动键与左Shift加速键，可快速调整视图位置；结合“Q/E”滚动键与“箭头+Shift”倾斜键，可精准调整视图角度，适配不同结构的展示需求。

- 常用功能快速调用：将高频使用的功能（如结构加载、高清渲染）添加至顶部工具栏快捷入口，无需每次通过右侧面板查找，减少操作步骤。
