# DAT

## volume dat 文件格式

> 支持 [gnuplot dat](http://www.gnuplot.info/docs_6.0/Gnuplot_6.pdf) 文件格式，内容为文本格式。

格式内容如下：

```txt 文件格式
#! 
x y z zdelta
x y z zdelta
x y z zdelta
x y z zdelta
x y z zdelta
x y z zdelta
x y z zdelta
```

格式解读：

第一行为 `#!` 开头，后面可以带其他内容

第二行开始格式为 `x y z zdelta` 或者是 `x y z zlow zhigh`
